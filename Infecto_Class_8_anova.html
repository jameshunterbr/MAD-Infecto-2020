Analysis of Variance - 2

Concepts

  • Focus on differences in group means
    • As opposed to regression
  • Balanced vs. Unbalanced Models
    • Same number of cases per group (balanced)
    • Different number of cases per group (unbalanced)
  • Multiple ANOVA model schemes
    • Class 6: One-Way ANOVA
    • Today: that + Two-Way ANOVA
  • Multiple Comparisons
    • Class 6: Bonferroni, Benjamini-Hochberg(BH)
    • Today: Tukey’s HSD
  • Use some functions today from car package

Example Data

  • New version of COVID clinical trial data
    • This time unbalanced
    • 72 cases:
      • 22 placebo, 26 nontussis, 30 hcq
      • 33 supplementary \(O_2\), 39 no supplementary \(O_2\)
  • Outcome measure is a scaled difference (between 1 and 10) of the difference between viral load at enrollment and viral load after 5 days of treatment

One-Way ANOVA

  • Focus on drug as factor variable
  • Two ways to execute in R
    • aov() + summary() of model
    • oneway.test() - only for one-way ANOVA
  • Check assumptions 1st
    • Equality of variances among groups:
      • car::leveneTest()
    • Normality of residuals
      • shapiro.test()
      • QQ-plot of residuals
      • Needs a calculated ANOVA model to have residuals

Check Equality of Variance Assumption

car::leveneTest(score ~ drug, data = clin_trial, center = mean)
## Levene's Test for Homogeneity of Variance (center = mean)

Multiple Comparisons

Why Do We Need Corrections?

  • I explained that when we do multiple tests
    • Elevated chance of having a at least p-value < \(\alpha\) by chance
  • If \(\alpha\) = 0.05 and 3 tests (as in this problem), at least 14% prob
    • If number of tests rises to 50 – at least a 92% prob of a significant result by chance

New Correction - Tukey’s HSD

  • Tukey’s Honest Significant Difference
  • Uses as input the aov() model
  • Works for balanced and mildly unbalanced designs
  • In base R, so has history and popularity (long time available)

Two-Way ANOVA

What is Different from One-Way?

  • Adding a second factor to be analyzed together
    • Here supplemental oxygen: o2
  • Equivalent to multiple linear regression with two covariates
  • Can also have interactions between the covariates

Visualize Differences for Supplemental Oxygen

gg_two_way <- clin_trial %>% 
  group_by(drug, o2) %>% 
  summarise(mean = mean(score),
            sd = sd(score),

Comments on ANOVA

  • aov() internally handles unbalanced models
  • If residuals very not normal, use kruskal.test() (Kruskal-Wallis test)
  • oneway.test() does not produce a model object that multiple comparisons can use
  • oneway.test() assumes model variables do not have equal variance
    • Must specify if they do
